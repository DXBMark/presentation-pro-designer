# PPTX Creation Workflow

Use when creating a new deck from a prompt, documents, data, images, or references.

1. Classify deck type.
2. Identify audience, purpose, source files, template, brand rules, and output requirements.
3. Ask clarification questions only if required.
4. Create a deck brief.
5. Create a slide-by-slide plan.
6. Select design tokens and slide patterns.
7. Build the PPTX using available host tools or implementation scripts.
8. Add charts, visuals, tables, speaker notes, and appendix where needed.
9. Render or inspect visually if possible.
10. Run QA.
11. Deliver file and summary.
