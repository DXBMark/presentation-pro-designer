# Infographic Visual Patterns

Use infographic patterns when complex information can be made clearer through structure.

## Pattern Selection

| Information Type | Pattern |
|---|---|
| list of concepts | icon cards, row list, grid cards |
| sequence or process | timeline, stepper, roadmap, flow |
| comparison | two-column, matrix, SWOT, pros/cons |
| hierarchy | tree, org chart, nested cards, mind map |
| cause and effect | fishbone, flow, causal chain |
| relationship | network, linked nodes, Venn-style explanation |
| metrics | KPI cards, mini charts, annotated table |

## Rules

- The visual must reduce cognitive load.
- Use icons consistently.
- Avoid decorative diagrams with no explanatory value.
- Keep text labels concise.
- Use template/brand colours first.
