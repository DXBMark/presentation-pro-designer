# PPTX Editing Workflow

Use when a PPTX is provided for editing, redesign, template reuse, content replacement, or visual improvement.

1. Inspect the PPTX.
2. Generate thumbnails where available.
3. Extract slide structure, layout patterns, colours, fonts, logos, footer rules, and repeated elements.
4. Decide whether to edit, duplicate layouts, or rebuild while preserving the design system.
5. Map new content to existing layouts.
6. Remove old placeholders and unused elements.
7. Clean unpacked PPTX parts if direct XML editing is used.
8. Render or inspect final slides.
9. Run QA and deliver.

## Editing Priorities

- Preserve brand/template unless redesign is requested.
- Preserve meaning unless the user asks for rewriting.
- Improve clarity, hierarchy, and visual balance.
- Avoid text overflow and hidden elements.
