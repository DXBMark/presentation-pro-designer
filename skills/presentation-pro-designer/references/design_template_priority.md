# Design Template Priority

If a PPTX template, sample deck, brand guide, or design system is supplied, it controls the visual design.

## Extract

- slide size and aspect ratio
- theme colours
- typography and font fallbacks
- title and section treatments
- margins, grids, and spacing
- logo placement and footer rules
- chart and table style
- image treatment
- repeated slide layouts

## Apply

- Preserve the template's design system unless the user asks for redesign.
- Use existing slide patterns before inventing new ones.
- Do not distort logos or recolour them unless the brand guide allows it.
- Use the same visual density and hierarchy as the template.
- If a required layout is missing, create a compatible layout using the extracted tokens.

## If No Template Is Provided

Use a clean, modern, accessible, presentation-safe default:
- 16:9 widescreen
- high contrast
- large readable typography
- restrained colour palette
- clear sectioning
- no decorative clutter
