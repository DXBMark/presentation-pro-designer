# Clarification Protocol

Ask questions only when the answer changes the deck structure, accuracy, source use, audience fit, financial integrity, or visual direction.

## Ask Up To Five Questions

Use targeted questions. Avoid long discovery questionnaires.

High-impact questions:
1. What type of presentation is this?
2. Who is the audience?
3. Which files/templates/brand standards should control the output?
4. Is this a new deck, a redesign, or an edit of an existing PPTX?
5. What deliverables are required?

## Proceed With Assumptions When

- The deck type is obvious from the prompt.
- Missing details are minor.
- The user asked for immediate execution.
- A safe default can be used.

List assumptions in the final summary.

## Ask Before Proceeding When

- User says “follow the attached standard” but no standard is visible.
- User says “use our brand” but no brand source is available.
- User asks for financial metrics without data or tool access.
- User asks for template compliance but no template is visible.
- Audience ambiguity would change tone, depth, or structure.
