## 🚀 How to Use

### Option 1: Quick Start (Minimal Setup)
1. Choose a quick example from the root `examples/` folder
2. Copy the text into your conversation with `@presentation-pro-designer`
3. Fill in the placeholder values marked with `[Insert ...]`
4. Attach any supporting files (templates, data, brand guides)
5. Run the prompt

### Option 2: Comprehensive Setup (Full Control)
1. Browse the `prompt-library/` folder
2. Find the prompt that matches your presentation type
3. Copy the entire prompt template
4. Customize every detail:
   - Audience and purpose
   - Source files and design authority
   - Required sections and content
   - Output format and speaker notes
5. Attach supporting files
6. Run the prompt with `@presentation-pro-designer`

### Option 3: Interactive Selection (Guided)
Not sure which prompt to use? Answer these questions:

**Q1: What type of presentation?**
- Education/training → `education-lesson-full.md` or `education-workshop-full.md`
- Company introduction → `corporate-company-profile-full.md`
- Pitching a service → `business-proposal-full.md`
- Showcasing a product → `product-service-full.md`
- Seeking investment → `investor-startup-deck-full.md` or `investor-public-company-deck-full.md`
- Proving company identity → `verification-profile-claim-full.md`
- Presenting data/results → `data-report-deck-full.md`
- Improving an existing deck → `improve-existing-pptx-full.md`
- Not sure? → `generic-presentation-full.md`

**Q2: Do you have supporting files?**
- Yes: Design template, brand guide, data, documents → Attach them
- No: The prompts will guide you to create them

**Q3: How much control do you want?**
- Just get it done → Use quick examples
- Full customization → Use comprehensive templates from `prompt-library/`

## 📋 Key Concepts

### Source Priority
All prompts follow this priority order:

1. **Your instructions** in the prompt
2. **Attached templates and brand guides** (highest design authority)
3. **Attached data and documents** (content sources)
4. **Project sources** you explicitly reference
5. **Default professional guidance** from the skill

### Clarification Rule
All prompts include guidance on when to ask questions vs. proceed with assumptions:
- Ask 5 specific questions when missing information materially changes the deck
- Proceed with reasonable assumptions for minor missing details
- Always list assumptions in the delivery summary

### Design Requirements
Every prompt includes:
- Brand consistency guidance
- Visual hierarchy principles
- Accessibility and readability standards
- Chart and table best practices
- Typography and spacing rules

### Speaker Notes
All templates request speaker notes that include:
- Presenter talking points
- Key explanations
- Timing guidance
- Question suggestions
- Common misconceptions or follow-up prompts

## 💡 Pro Tips

### For Best Results:

1. **Gather your sources first**
   - Templates, brand guides, data, documents
   - Attach them before running the prompt

2. **Be specific about audience and purpose**
   - Who will see this presentation?
   - What action or understanding should it create?

3. **Specify your design authority**
   - If you have a template, say it's the "highest design priority"
   - If you don't have one, say "create a clean professional style"

4. **Mark missing data clearly**
   - Instead of inventing numbers, mark them as "DATA NEEDED"
   - The prompt will create placeholders for you

5. **Ask for what you need**
   - Speaker notes? Ask for them
   - Executive version? Request it
   - Charts instead of tables? Specify it

### Template Adaptation:

You can copy any full prompt and:
- Remove sections you don't need
- Add custom sections for your needs
- Adjust the slide count
- Change audience or purpose
- Combine ideas from multiple templates
