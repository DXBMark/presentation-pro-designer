# Startup Investor Deck Prompt

Use this prompt to create a startup pitch deck for fundraising, investor meetings, accelerators, or demo days.

```text
@presentation-pro-designer

Create a professional startup investor pitch deck.

Presentation type:
Startup investor deck.

Company / startup:
[Insert company name]

Funding context:
[Pre-seed / seed / Series A / strategic investment / demo day / accelerator]

Purpose:
Create a clear investor deck that explains the problem, solution, market opportunity, product, business model, traction, team, financials, and investment ask.

Audience:
[Angel investors / VCs / strategic partners / accelerator panel]

Slide count:
[10–14 slides, or write: choose the best number based on content]

Source priority:
1. Follow this prompt.
2. Use attached business plan, pitch notes, financials, product screenshots, market research, or founder notes as the main source.
3. Use attached brand guide or PPTX template as the highest design priority.
4. Do not use external sources unless I approve.
5. Do not invent traction, revenue, market size, customer logos, financials, partnerships, or investor claims.

Required structure:
- Title slide
- Problem
- Solution
- Product / demo overview
- Target customer
- Market opportunity
- Business model
- Traction or validation, only if provided
- Go-to-market strategy
- Competition / positioning
- Team
- Financials or projections, only if provided
- Investment ask and use of funds
- Closing slide

Content requirements:
- Make the story investor-ready and concise.
- Use direct headlines that communicate the key point of each slide.
- Quantify claims only when data is provided.
- Mark missing financial or traction data as "DATA NEEDED".
- Use simple, credible language.
- Avoid hype and unsupported market claims.

Financial requirements:
- If financial data is provided, summarise revenue, costs, margins, runway, burn, and forecast clearly.
- If financial data is missing, include a placeholder slide with "DATA NEEDED" and list what is required.
- Do not fabricate numbers.

Design requirements:
- Premium, modern, clean, and brand-consistent.
- Use strong visual hierarchy.
- Use diagrams, traction charts, market maps, and tables where useful.
- Keep text concise.
- Ensure charts are readable and clearly labelled.

Speaker notes:
Include speaker notes with concise investor-facing talking points for each slide.

Clarification rule:
If the funding stage, audience, financial data, traction, or investment ask is unclear, ask up to 5 specific questions before creating the deck.

Output:
- Create a PPTX file.
- Include a short summary of sources used, assumptions made, missing data, and QA checks completed.
```
