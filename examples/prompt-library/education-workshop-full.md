# Education Workshop / CPD Deck Prompt

Use this prompt for teacher training, CPD, staff development, or education workshops.

```text
@presentation-pro-designer

Create a professional education workshop PowerPoint presentation.

Presentation type:
Education / teacher training / CPD workshop.

Workshop title:
[Insert workshop title]

Audience:
[Teachers / teaching assistants / department staff / school leadership / trainee teachers]

Purpose:
Create a professional workshop deck that explains the topic, engages participants, supports discussion, and gives staff practical takeaways they can apply.

Duration:
[Insert duration, e.g. 30 minutes / 60 minutes / half-day]

Slide count:
[Insert number, or write: choose the best number based on the workshop flow]

Source priority:
1. Follow this prompt.
2. Use any attached policy, framework, school standard, research summary, or training document as the main source.
3. Use the attached PowerPoint template or brand guide as the highest design priority.
4. Use only uploaded or project sources unless I approve external sources.

Required structure:
- Title slide
- Why this topic matters
- Session aims
- Current challenge or context
- Key framework or model
- Practical examples
- Participant activity or discussion task
- Reflection prompt
- Implementation checklist
- Next steps
- Optional appendix/resources slide

Content requirements:
- Make the content practical and relevant to teachers.
- Avoid generic CPD language.
- Use examples that connect directly to classroom practice.
- Include participant interaction every 10–15 minutes where appropriate.
- Include clear takeaways.
- Avoid unsupported claims or vague research references.

Design requirements:
- Professional, clean, school-appropriate, and accessible.
- Use strong visual hierarchy.
- Use diagrams, process flows, cards, and examples where useful.
- Avoid overloaded slides.
- Keep text readable from the back of a room.

Speaker notes:
Include notes for the facilitator:
- key talking points
- questions to ask
- suggested timings
- activity instructions
- likely participant responses
- follow-up prompts

Clarification rule:
If the audience, duration, policy context, or workshop outcome is unclear, ask up to 5 specific questions before creating the deck.

Output:
- Create a PPTX file.
- Include a short summary of sources used, assumptions made, design choices, and QA checks completed.
```
