# Public Company Investor / Equity Analysis Deck Prompt

Use this prompt for investor decks based on public company analysis, equity research, financial performance, valuation, peer comparison, or investment thesis.

```text
@presentation-pro-designer

Create a professional public company investor presentation.

Presentation type:
Investor deck / equity analysis presentation.

Company:
[Insert company name and ticker]

Purpose:
Create a decision-ready investor presentation that summarises the company's financial position, growth trends, investment thesis, valuation considerations, risks, and peer positioning.

Audience:
[Portfolio manager / analyst / investment committee / finance team / business audience]

Slide count:
[Insert number, or write: choose the best number based on analysis depth]

Source priority:
1. Follow this prompt.
2. Use attached financial data, filings, spreadsheet, analyst notes, company presentation, or research document as the main source.
3. If a financial data tool is available, use it only if I explicitly approve.
4. Do not use external sources unless I approve.
5. Do not invent prices, target prices, estimates, ratings, valuation multiples, market caps, or financial figures.

Required structure:
- Title slide
- Executive summary
- Company snapshot
- Investment thesis
- Revenue and profitability overview
- Financial growth trends
- Balance sheet and cash flow highlights
- Segment or geographic mix, if data is available
- Peer comparison, if data is available
- Valuation framework, if data is available
- Key risks
- Scenario analysis, if assumptions are provided
- Conclusion / recommendation
- Appendix with source tables

Financial analysis requirements:
- Use provided or retrieved data only.
- Mark missing data as "DATA NEEDED".
- Clearly cite the source and date of every major financial number.
- Use tables for income statement, balance sheet, cash flow, estimates, and peer comparison.
- Use charts only when the data supports them.
- Highlight trends, not just numbers.

Analysis logic:
- Identify growth acceleration or deceleration.
- Compare revenue growth with profit growth.
- Check whether free cash flow supports net income.
- Identify balance sheet risks if liabilities or net debt are material.
- Avoid investment recommendations unless enough data is provided.

Design requirements:
- Institutional, clean, and finance-ready.
- Use restrained colours, high-contrast charts, and clear tables.
- Avoid decorative visuals.
- Use direct slide titles that state the insight.

Speaker notes:
Include speaker notes with analyst-style talking points and caveats.

Clarification rule:
If the ticker, audience, financial source, investment objective, or valuation method is unclear, ask up to 5 specific questions before creating the deck.

Output:
- Create a PPTX file.
- Include a short summary of sources used, assumptions made, DATA NEEDED items, and QA checks completed.
```
