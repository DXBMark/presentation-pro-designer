# Education Lesson Deck Prompt

Use this prompt when you want to create a complete classroom lesson PowerPoint.

```text
@presentation-pro-designer

Create a professional PowerPoint lesson deck.

Presentation type:
Education / classroom lesson.

Topic:
[Insert lesson topic]

Subject:
[Insert subject]

Audience:
[Insert year group, age range, prior knowledge level]

Purpose:
Create a student-facing lesson deck that explains the topic clearly, supports classroom delivery, and includes activities, checks for understanding, and teacher notes.

Duration:
[Insert lesson length, e.g. 45 minutes / 50 minutes / 60 minutes]

Slide count:
[Insert number, or write: choose the best number based on the lesson flow]

Source priority:
1. Follow my instructions in this prompt.
2. Use the attached syllabus, scheme of work, lesson notes, or standards as the main content source.
3. Use the attached PowerPoint template or brand guide as the highest design priority.
4. Use only uploaded files and project sources unless I explicitly approve external sources.
5. Do not invent curriculum requirements, standards, facts, or references.

Required structure:
- Title slide with clear lesson focus
- Retrieval practice or starter activity
- Learning objectives and success criteria
- Key vocabulary slide, if useful
- Main concept explanation slides
- Worked example
- Guided practice task
- Independent task
- Misconception check
- Check for understanding
- Exit ticket
- Optional homework or extension task

Content requirements:
- Use student-friendly language.
- Explain one idea per slide.
- Break complex ideas into small, teachable steps.
- Include examples before independent practice.
- Include questions that check understanding.
- Include challenge or extension where appropriate.
- Avoid text-heavy slides.

Design requirements:
- Follow the attached design template if provided.
- Keep the deck clean, modern, accessible, and classroom-readable.
- Use high-contrast colours.
- Use large readable text suitable for projection.
- Use high-quality visuals, icons, diagrams, or simple illustrations only when they support learning.
- Avoid clutter, stretched images, tiny text, and decorative visuals that do not support understanding.

Speaker notes:
Include speaker notes for each slide with:
- what the teacher should say
- questions to ask
- expected student answers
- timing guidance
- common misconceptions
- differentiation or extension suggestions

Clarification rule:
If important information is missing, ask up to 5 specific questions before creating the deck. If the missing information is minor, make reasonable assumptions and list them in the final summary.

Output:
- Create a PPTX file.
- Include a short summary of sources used, assumptions made, design decisions, and QA checks completed.
```
