# Improve Existing PowerPoint Prompt

Use this prompt when you already have a PPTX and want it improved, redesigned, cleaned, restructured, or made more professional.

```text
@presentation-pro-designer

Improve the attached PowerPoint presentation.

Presentation type:
PPTX editing / redesign / improvement.

Goal:
Make the presentation clearer, more professional, more visually consistent, and better aligned with its audience and purpose.

Audience:
[Insert audience]

Purpose:
[Insert purpose of the presentation]

Editing mode:
[Choose one]
- Light polish: keep structure, improve design and clarity.
- Redesign: keep content, significantly improve layouts and visuals.
- Restructure: reorganise content and slide flow.
- Template match: adapt the deck to the attached template.
- Executive version: make it shorter and more decision-ready.
- Teaching version: make it more suitable for classroom delivery.

Source priority:
1. Follow this prompt.
2. Preserve the original meaning unless I ask for rewriting.
3. Follow any attached template, brand guide, or sample deck as the highest design priority.
4. Use only uploaded or project sources unless I approve external sources.
5. Do not add unsupported claims.

Required improvements:
- Improve slide structure and flow.
- Reduce text-heavy slides.
- Improve titles so they communicate the key message.
- Improve spacing, alignment, hierarchy, and consistency.
- Replace weak layouts with stronger slide patterns.
- Improve tables and charts where needed.
- Fix stretched, blurry, or low-quality visuals.
- Remove unnecessary decoration.
- Ensure readability and accessibility.

Optional additions:
- Add speaker notes if useful.
- Add section dividers if needed.
- Add summary or next steps slide if missing.
- Add charts or diagrams only if supported by the content.

Design requirements:
- Follow the existing template if it is clearly intentional.
- If the current design is inconsistent, create a clean and coherent style.
- Use brand colours and fonts where available.
- Keep all slides readable.
- Avoid tiny text, poor contrast, overcrowding, and inconsistent icon styles.

Clarification rule:
If the desired editing mode, audience, template priority, or allowed content changes are unclear, ask up to 5 specific questions before editing.

Output:
- Return an edited PPTX file.
- Include a short summary of what changed, what was preserved, assumptions made, and QA checks completed.
```
