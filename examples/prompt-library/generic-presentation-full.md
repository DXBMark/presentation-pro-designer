# Generic Professional Presentation Prompt

Use this prompt when the topic does not clearly fit one category.

```text
@presentation-pro-designer

Create a professional PowerPoint presentation.

Presentation type:
Generic professional presentation.

Topic:
[Insert topic]

Audience:
[Insert audience]

Purpose:
[Explain what the presentation should achieve]

Duration:
[Optional]

Slide count:
[Insert number, or write: choose the best number based on content]

Source priority:
1. Follow this prompt.
2. Use attached files as the main content source.
3. Use attached PowerPoint template, brand guide, or sample deck as the highest design priority.
4. Use only uploaded and project sources unless I approve external sources.
5. Do not invent unsupported facts, claims, figures, or references.

Before creating the deck:
If the type of presentation is unclear, classify it as one of the following:
- education
- corporate
- business proposal
- product/service presentation
- brand/company introduction
- verification/profile claim
- investor deck
- data report
- generic presentation

If classification is unclear or affects the deck structure, ask up to 5 specific questions.

Required structure:
Choose the most suitable slide structure based on the presentation type and audience. At minimum, include:
- Title slide
- Context or overview
- Main content sections
- Examples, evidence, or supporting details
- Summary
- Next steps or closing slide

Design requirements:
- Clean, professional, modern, accessible, and audience-appropriate.
- Follow any provided template or brand guide.
- Use high-quality visuals.
- Use diagrams, charts, or tables only where they improve clarity.
- Avoid clutter, tiny text, stretched images, and unsupported claims.

Speaker notes:
Include speaker notes if useful for delivery.

Output:
- Create a PPTX file.
- Include a short summary of sources used, assumptions made, presentation type selected, design decisions, and QA checks completed.
```
