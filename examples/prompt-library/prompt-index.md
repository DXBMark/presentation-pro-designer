## Best Practice

For strongest results, always include:

1. Presentation type
2. Topic
3. Audience
4. Purpose
5. Source files to use
6. Design template or brand guide, if available
7. Slide count or duration
8. Output format
9. Whether speaker notes are required
10. Whether external sources are allowed

## Source Rule

If a file, template, syllabus, standard, brand guide, dataset, or verification document is attached, state clearly whether it should be treated as:

- main content source
- design authority
- supporting reference
- data source
- style reference
- file to edit
